# 📊 Global Business Performance Dashboard

## Overview
An interactive financial performance dashboard 
built using Tableau, analyzing Global Superstore 
sales data across 147 countries (2011-2014).

## Business Questions Answered
- Which markets are most/least profitable?
- How does discounting impact profitability?
- Which product categories drive maximum loss?
- What is the overall business growth trend?

## Key Insights
1. Tables sub-category has highest loss (-$64,083)
   despite significant sales volume
2. High discount (30%+) directly causes negative 
   profit across all categories
3. USA & Australia are top profit-generating 
   markets globally
4. Business grew consistently — Sales reached 
   $4M by 2014

## Dashboard Components
- Global Profit Map (147 countries)
- Yearly Sales & Profit Trend (2011-2014)
- Profit & Loss by Sub-Category
- Impact of Discount on Profit

## Tools & Dataset
- Tool: Tableau Desktop
- Dataset: Global Superstore (51,000+ records)
- Skills: Data Visualization, Business Analysis
